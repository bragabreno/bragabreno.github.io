A representação unária

Uma noção bastante natural de contagem é aquela em que se associa um único símbolo à(s) ocorrência(s) de um objeto ou evento[1], de modo que se mantenha registro da quantidade de vezes em que eles acontecem.

Quer dizer, considere o símbolo \(|\). De início, enquanto não houver contagem, não há nada escrito. Por convenção, ter-se-ia ' ' (sem as aspas, isto é, um espaço em branco). No entanto, à medida em que certo evento ou objeto ocorre, que se escreva \(|\).

Caso já exista uma cadeia escrita (isto é, caso já houver registro de realizações do evento/objeto), basta escrever \(|\), concatenando o símbolo escrito à cadeia. Assim, tem-se:

\[|,\ ||,\ |||,\ ||||,\ ...\]

A essa representação, chamamos representação numérica unária. E, como já foi mencionado, é uma notação que descreve bem a noção intuitiva de contagem (é aplicada quando contamos com os dedos, por exemplo).

Além disso, também é possível traçar uma relação direta entre essa representação e a noção formal de um número natural. É o que se mostrará a seguir, através de uma enunciação (não tão rigorosa, mas suficiente para os nossos fins) dos Axiomas de Peano, que determinam o conjunto \( \mathbb{N} \) dos números naturais.

  • Axioma 1: Há um número natural chamado zero;
  • Axioma 2: Há uma função \(S: \mathbb{N} \rightarrow \mathbb{N}\) chamada "sucessor", de modo que, se \(n\) é um número natural, então também \(S(n)\), o sucessor de \(n\), o é;
  • Axioma 3: Para qualquer número natural \(n\), o zero não pode ser representado como \(S(n)\). Isto é, o zero não é sucessor de nenhum outro número;
  • Axioma 4: Para quaisquer números naturais \(m\) e \(n\), se \(S(n) = S(m)\), então \(n = m\);
  • Axioma 5: Dado um certo conjunto \(X\), se o zero pertence a \(X\) e, também, \(S\) pertence a \(X\), então \(X\) contém todos os números naturais.

Definição de notação

Para que usemos a notação simbólica (proposta inicialmente) a fim de nos referirmos à sistematização de Peano, seguem-se as seguintes definições[2]:

  • \(\ \ := zero\);
  • \(|\ := S(zero)\);
  • \(||:= S(S(zero))\);
  • \(|||:= S(S(S(zero)))\);
  • \(\vdots\)
  • \(|^n\ := S^n(zero)\);
  • \(\vdots\)

Operações básicas

Agora, introduziremos os axiomas que definem as operações básicas \(+\) (adição) e \(\cdot\) (multiplicação), que são essenciais para que se desenvolvam construções mais complexas com os números naturais:

  • A1: Para todo número natural \(n\), \(n + zero = n\);
  • A2: Para quaisquer números naturais \(m\), \(n\), \(m + S(n) = S(m + n)\);
  • M1: Para todo número natural \(n\), \(n \cdot zero = zero\);
  • M2: Para quaisquer números naturais \(m\), \(n\), \(m \cdot S(n) = (m \cdot n) + m\).

Exemplos de operações

Dessa maneira, já conseguimos realizar operações fazendo uso da notação que definimos:

Soma
\[| + || = S(zero) + S(S(zero))\]
\[= S(S(zero) + S(zero))\]
\[= S(S(S(zero) + zero))\]
\[= S(S(S(zero))) = |||\]
Multiplicação
\[|| \cdot ||| = S(S(zero)) \cdot S(S(S(zero)))\]
\[= (S(S(zero)) \cdot S(S(zero))) + S(S(zero))\]
\[= ((S(S(zero)) \cdot S(zero)) + S(S(zero))) + S(S(zero))\]
\[= (((S(S(zero)) \cdot zero) + S(S(zero))) + S(S(zero))) + S(S(zero))\]
\[= (((zero) + S(S(zero))) + S(S(zero))) + S(S(zero))\]
\[= S(S(S((S(S(zero))))) + S(zero))\]
\[= S(S(S(S((S(S(zero))))) + zero))\]
\[= S(S(S(S((S(S(zero))))))) = ||||||\]

  2. Aqui, \(:=\) denota "está definido como". Quer dizer, em todas as ocorrências dos símbolos à esquerda de \(:=\), eles têm o mesmo significado.
